### install **Rust**

<hr>

- I installed it with the help of Scoop (Windows)!

```shell
scoop install rustup
```

- You can also use Scoop to upgrade it

```shell
scoop upgrade rustup
```

- You can also use Scoop to uninstall it

```shell
scoop uninstall rustup
```
